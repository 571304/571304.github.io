Hexo+NEXT主题之博客优化

发表于 2017-12-07

说明

hexo版本：3.4.3
主题：NEXT主题
主题版本：5.0.1

安装/更新 主题

在终端切换到hexo 根目录. 在hexo目录下一定有 node_modules, source, themes 和其他文件夹:

cd hexo
ls
_config.yml  node_modules  package.json  public  scaffolds  source  themes

下载[最新的 master 分支]可能会 不稳定, 但是包含最新的特色，推荐开发者下载.

mkdir themes/next
curl -L https://api.github.com/repos/iissnan/hexo-theme-next/tarball | tar -zxv -C themes/next --strip-components=1

更新主题

cd themes/next
git pull

启用NexT主题

在 hexo 根目录下 的配置文件_config.yml里设置主题:

theme: next

在站点配置文件_config.yml中可以将语言切换成中文

language: zh-Hans

配置NexT主题

添加页面 Page

添加一个标签页面，里面包含您网站中的所有标签。

创建一个名为 tags 页面

hexo new page tags

编辑标签页, 设置页面类型为 tags

title: All tags
date: 2014-12-22 12:39:04
type: tags
添加一个分类页面，里面包含您网站中的所有分类。

创建一个名为 categories 页面

hexo new page categories

编辑标签页, 设置页面类型为 categories

title: All categories
date: 2014-12-22 12:39:04
type: categories
添加一个关于页面，里面包含个人信息。

创建一个名为 about 页面

hexo new page about
添加一个图片页面，用于展示图片。

创建一个名为 photos 页面

hexo new page photos

编辑标签页, 设置页面类型为 photos

title: photos
date: 2014-12-22 12:39:04
type: photos

添加页面到主题

修改主题配置文件 _config.yml 里
menu:
  home: / || home
  about: /about/ || user
  tags: /tags/ || tags
  categories: /categories/ || th
  archives: /archives/ || archive
添加本地搜索

执行命令

npm install hexo-generator-search --save

修改主题配置文件 _config.yml 里enable值为true 开启搜索功能

local_search:
  enable: true

RSS订阅

执行命令

npm install hexo-generator-search --save

设置头像：

将选中的图片文件放到 hexo/themes/next/source/images目录下，命名为avatar.png，然后编辑站点配置文件，新增字段

avatar: /images/avatar.png

阅读全文按钮：

方法一：编写文章时，在文章中插入<!– more –>，在哪里插入就从哪里截断

方法二：编辑主题配置文件，修改字段：将enable值设为true，之后每篇文章会自动在150处截断

auto_excerpt:
  enable: false
 length: 150（可自行设置）

关闭换行符警告：

执行hexo d命令时，会产生警告如下：warning: LF will be replaced by CRLF
执行命令

git config --global core.autocrlf false

false表示取消自动转换功能。适合纯Windows
true表示提交代码时把CRLF转换成LF，签出时LF转换成CRLF。适合多平台协作
input表示提交时把CRLF转换成LF，签出时不转换。适合纯Linux或Mac
